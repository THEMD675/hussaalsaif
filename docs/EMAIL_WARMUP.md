## Why Warmup Is Required

A brand-new sending domain has zero reputation with Gmail, Outlook, Yahoo, and other inbox providers. If you start blasting 50 emails on day one, they will land in spam or get silently dropped. Warmup builds sender reputation gradually by demonstrating to inbox providers that:

1. Real humans send from this domain
2. Recipients open, read, and reply to these emails
3. No one marks them as spam
4. Authentication (SPF, DKIM, DMARC) passes

**Minimum warmup period:** 4 weeks before any cold outreach.

## Subject Line Best Practices

### DO:
- Keep subject lines under 50 characters
- Use the brand's name: "Collaboration Inquiry -- Hussa AlSaif x [Brand]"
- Be specific: "Curly Hair Campaign Proposal for [Brand]"
- Use title case
- Personalize with recipient's name when possible: "[Name], quick question about [Brand] influencer program"

### DO NOT:
- ALL CAPS: "COLLABORATION OPPORTUNITY!!!"
- Exclamation marks: "Amazing Partnership Idea!"
- Spam trigger words: "Free," "Guaranteed," "Act Now," "Limited Time," "Click Here"
- Emojis in subject lines (some spam filters flag these on new domains)
- Generic subjects: "Partnership," "Hello," "Inquiry"

### Proven Subject Lines for Brand Outreach:
1. `Collaboration Inquiry -- Hussa AlSaif x [Brand Name]`
2. `Partnership Opportunity -- Saudi Curly Hair Creator x [Brand]`
3. `[Name], loved [Brand]'s [specific campaign/product]`
4. `Quick question about [Brand]'s GCC influencer program`
5. `Following up -- Hussa AlSaif x [Brand] collaboration`
6. `[Brand] x Hussa AlSaif -- media kit attached`

## If Things Go Wrong

| Symptom | Likely Cause | Fix |
|---------|-------------|-----|
| Emails going to spam | Domain too new, sending too fast | Reduce volume to 5/day for 1 week. Ask recipients to mark "Not Spam" |
| Low open rates (<20%) | Subject lines or spam placement | Test with mail-tester.com. Revise subject lines. Reduce volume. |
| High bounce rate (>3%) | Bad email addresses | Remove bounced addresses. Only email verified contacts. |
| Gmail shows "Be careful with this message" | Missing DKIM or SPF fail | Re-check DNS records in Vercel. Re-verify in Resend. |
| Resend returns errors | API key issue or domain not verified | Check Vercel env vars. Check Resend dashboard domain status. |
| Google Postmaster shows spam rate >0.3% | Content triggers or list quality | Pause all cold outreach. Send only to known contacts for 1 week. Review email content for spam triggers. |
