## 2. PIPELINE STAGES

Every brand deal moves through these stages. Track each one in the CRM.

| Stage | Definition | Action Required | Typical Duration |
|-------|-----------|----------------|-----------------|
| **Lead** | Brand identified as a target, no contact made yet | Research contact person, prepare personalized pitch | 1-3 days |
| **Outreach** | Initial email or DM sent | Wait for response, prepare follow-up for Day 5-7 | 5-7 days |
| **Response** | Brand replied (positive, neutral, or requesting info) | Send media kit / rate card, answer questions | 1-3 days |
| **Negotiation** | Terms being discussed (deliverables, pricing, timeline, usage rights) | Negotiate rate, define scope, draft agreement terms | 3-14 days |
| **Contracted** | Agreement signed by both parties | Confirm deliverables, timeline, payment terms in writing | 1-3 days |
| **In Production** | Content being created (shooting, editing, review cycles) | Create content, submit for brand approval, manage revisions | 7-30 days |
| **Delivered** | Content posted on agreed platforms | Share analytics/screenshots with brand, confirm posting | 1-2 days |
| **Complete** | Payment received, campaign fully closed | Log final metrics, request testimonial, archive deal | 1-30 days (net payment terms) |
| **Follow-up** | Repeat business opportunity identified | Reach out for next campaign, propose retainer/ambassador | Ongoing (quarterly check-in) |

### Stage Movement Rules

- A deal can only move forward, never backward (if a deal falls through during Negotiation, mark it "Lost" with a reason)
- Add a "Lost" status for deals that die at any stage -- always log the reason
- Add a "Paused" status for deals temporarily on hold
- Every deal in Outreach must have a follow-up date set
- Every deal in Negotiation must have a proposed value logged
- Every deal in Contracted must have deliverables and deadline confirmed

## 5. CRM SETUP INSTRUCTIONS

### Step-by-Step: Notion CRM

**Time required: 30 minutes**

1. **Create a Notion account** at notion.so (free plan works)

2. **Install the Influencer CRM template** from notion.com/templates/influencer-crm

3. **Create these database properties (columns):**

| Property | Type | Options/Format |
|----------|------|---------------|
| Brand Name | Title | -- |
| Contact Person | Text | First name, last name |
| Contact Email | Email | -- |
| Contact Phone | Phone | -- |
| Company / Agency | Text | Parent company or PR agency |
| Deal Stage | Select | Lead, Outreach, Response, Negotiation, Contracted, In Production, Delivered, Complete, Follow-up, Lost, Paused |
| Deal Value (SAR) | Number | Currency format |
| Deal Value (USD) | Number | Currency format |
| Platform | Multi-select | Instagram, TikTok, YouTube, Snapchat, Event, Multi-platform |
| Deliverables | Text | Specific content to be created |
| Deadline | Date | Content delivery date |
| Payment Status | Select | Not invoiced, Invoiced, Paid, Overdue |
| Payment Date | Date | When payment was received |
| Outreach Date | Date | When first contact was made |
| Follow-up Date | Date | Next follow-up action required |
| Notes | Text | Any additional context |
| Priority | Select | High, Medium, Low |
| Category | Select | Beauty, Haircare, Fashion, Lifestyle, Tech, Retail, Fragrance |

4. **Create these views:**
   - **Pipeline Board** -- Kanban view grouped by Deal Stage (primary working view)
   - **Calendar** -- Calendar view by Deadline (see upcoming deliverables)
   - **Follow-up List** -- Table view filtered to deals where Follow-up Date = today or past
   - **Revenue Tracker** -- Table view sorted by Deal Value, filtered to Contracted + In Production + Delivered
   - **Completed Archive** -- Table view filtered to Complete stage

5. **Import past deals** from the table in Section 3 above

6. **Import target brands** from Section 4 as Lead-stage entries

### Step-by-Step: HubSpot Free (Alternative)

If you prefer HubSpot:

1. Sign up at hubspot.com/products/crm (no credit card needed)
2. Link email provider (Gmail/Outlook)
3. Go to Deals > Pipelines
4. Create pipeline stages matching Section 2
5. Add custom properties: Platform, Deliverables, Category
6. Upload contacts via CSV
7. Install HubSpot mobile app

## 6. PIPELINE OPERATING PROCEDURES

### Weekly Rhythm

| Day | Action |
|-----|--------|
| **Sunday** | Review pipeline board. Move any stalled deals. Send follow-ups for Outreach stage deals past 7 days. |
| **Monday** | Send 3-5 new outreach emails to Lead-stage brands. |
| **Tuesday** | Follow up on Response-stage deals. Advance negotiations. |
| **Wednesday** | Content production check-in. Review In Production deals for deadline compliance. |
| **Thursday** | Send 3-5 new outreach emails. Review any brand responses received. |
| **Saturday** | Weekly pipeline review: log deal values, update stages, plan next week's targets. |

### Monthly Review

- Count: deals created, deals won, deals lost, total pipeline value
- Calculate: close rate (won / total outreach), average deal value, time-to-close
- Identify: which outreach templates get the most responses, which brand categories convert best
- Adjust: shift outreach focus to highest-converting categories
